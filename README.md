## âœ¨ Ce face aplicatia?

1.  **Dashboard Modern:** Vezi rapid cate fisiere ai scanat, cate duplicate s-au gasit si folderul curent.
2.  **Organizare Automata:**
    * **Standard:** Muta fisierele in foldere dupa tip (ex: Imagini, Documente, Video).
    * **Smart:** Clasificare inteligenta.
    * **Custom:** Iti faci propriile reguli (ex: daca numele contine "factura", muta in folderul "Contabilitate").

4.  **Istoric:** Tot ce muta aplicatia este salvat intr-o baza de date, ca sa stii mereu unde au ajuns fisierele tale.
5.  **Cautare:** Filtreaza rapid lista de fisiere scanate.
